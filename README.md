# 💈 Black Gold Barbearia — Landing Page

Landing page moderna, responsiva e altamente persuasiva desenvolvida para uma barbearia premium, com foco em conversão via WhatsApp.

---

## 🚀 Objetivo

Este projeto foi criado com o objetivo de:

- Gerar leads de forma rápida
- Converter visitantes em clientes
- Apresentar serviços de forma profissional
- Facilitar o agendamento via WhatsApp

---

## 🧠 Tecnologias utilizadas

- HTML5
- CSS3 (custom properties, responsividade e animações)
- JavaScript (interações e animações)
- Google Fonts

---

## 🎯 Funcionalidades

- ✅ Design moderno e premium
- ✅ Totalmente responsivo (mobile-first)
- ✅ Botões de WhatsApp funcionais
- ✅ Animações suaves (scroll e reveal)
- ✅ Parallax no hero
- ✅ Navbar dinâmica ao scroll
- ✅ Seções estratégicas para conversão:
  - Hero (chamada forte)
  - Serviços
  - Diferenciais
  - Depoimentos
  - CTA final

---

## 📱 Responsividade

O site foi desenvolvido com foco principal em dispositivos móveis, garantindo:

- Boa usabilidade em celulares
- Layout adaptável em tablets e desktops
- Performance leve e rápida
